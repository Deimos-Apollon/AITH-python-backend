# Описание ручек

## GET ручки

* /factorial?n={n}
  В качестве параметра n ожидается целое неотрицательное число.
* /fibonacci?n={n}
  В качестве параметра n ожидается целое неотрицательное число.
* /mean?data={data}
  В качестве параметра data ожидается последовательность чисел, разделенных через "," (например, data=1,2,3) или одно число.
